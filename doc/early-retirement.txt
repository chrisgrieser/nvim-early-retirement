1. nvim-early-retirement ðŸ‘´ðŸ‘µ*early-retirement-nvim-early-retirement-ðŸ‘´ðŸ‘µ*

Send buffers into early retirement by automatically closing them after x
minutes of inactivity.

Makes the bufferline or `:bnext` less crowded.


- |early-retirement-installation|
- |early-retirement-configuration|
- |early-retirement-credits|

CONFIGURATION  *early-retirement-nvim-early-retirement-ðŸ‘´ðŸ‘µ-configuration*

>lua
    -- default values
    opts = {
        -- if a buffer has been inactive for this many minutes, close it
        retirementAgeMins = 20,
    
        -- filetypes to ignore
        ignoredFiletypes = {},
    
        -- will not close the alternate file
        ignoreAltFile = true,
    
        -- will ignore buffers with unsaved changes. If false, the buffers will
        -- automatically be written and then closed.
        ignoreUnsavedChangesBufs = true,
    
        -- ignore non-empty buftypes, e.g. terminal buffers
        ignoreSpecialBuftypes = true,
    
        -- ignore visible buffers (buffers open in a window, "a" in `:buffers`)
        ignoreVisibleBufs = true,
    
        -- uses vim.notify for plugins like nvim-notify
        notificationOnAutoClose = false,
    }
